# Перенос лендинга в Tilda поблочно

Каждый файл — самодостаточный HTML-блок (своя вёрстка + стили + скрипты). Вставляется в Tilda в блок **«Код»** (вкладка «Код» в редакторе vibe-блока, который на твоём скрине), либо в стандартный блок **T123 (HTML-код)** / Zero Block → элемент HTML.

## Порядок блоков (как на оригинале)

| # | Файл | Что это |
|---|------|---------|
| 1 | `01-header.html` | Шапка: sticky-меню + мобильное бургер-меню |
| 2 | `02-hero.html` | Первый экран: заголовок, триггеры, кнопки, фото |
| 3 | `03-trust.html` | Опыт в цифрах + цитата юриста |
| 4 | `04-services.html` | 5 направлений практики с раскрытием доп. услуг |
| 5 | `05-pricelist.html` | Прайс-лист разовых услуг |
| 6 | `06-cases.html` | Кейсы «было → стало» |
| 7 | `07-documents.html` | Документы и отзывы — карусель |
| 8 | `08-cta.html` | Призыв к действию (тёмный блок) |
| 9 | `09-footer.html` | Подвал: контакты, соцсети, правовая инфо |

## Как вставлять

1. В Tilda создаёшь vibe-блок → вкладка **«Код»**.
2. Открываешь нужный `.html`, копируешь **всё содержимое** файла, вставляешь, жмёшь «Сохранить».
3. Повторяешь для каждого блока в порядке из таблицы.

## Важные детали

- **Дизайн-токены** (цвета/шрифты) вшиты в каждый блок как CSS-переменные на обёртке (`--bg`, `--paper`, `--fg`, `--muted`, `--bd`, `--sage`, `--sage-d`). Это аналог `design-tokens.ts` для standalone-HTML. Менять цвет — в одном месте вверху `<style>` блока.
- **Шрифты**: Cormorant Garamond + Inter подключаются через Google Fonts (`@import` в каждом блоке, браузер дедуплицирует).
- **Скролл к секциям**: ссылки в шапке ведут на якоря `#razvod`, `#alimenty`, `#imushchestvo`, `#deti`, `#zemlya`, `#uslugi`, `#cases`. Эти `id` уже стоят на соответствующих блоках — навигация работает между Tilda-блоками на одной странице.
- **Фото портрета** (Hero): замени `src` в `02-hero.html` на ссылку загруженного в Tilda изображения (или вставь картинку отдельным элементом Tilda поверх).
- **Контакты** заданы переменными в коде — телефон `+7 (914) 123-45-67`, WhatsApp `wa.me/79141234567`, Telegram `t.me/sysuev_lawyer`. Меняй прямо в `href`.
- **CSS изолирован** под префиксом `.qlh-*`, классы не конфликтуют со стилями Tilda.
- **Цели Яндекс.Метрики**: в оригинале на клики висели заглушки `handleGoalTrigger`. В HTML-блоках их нет — при необходимости навесь `onclick="ym(XXXXXX,'reachGoal','...')"` на нужные кнопки.

## Оригинальный проект (React)

Запуск локально:

```bash
npm install --legacy-peer-deps
npm run dev   # http://localhost:8080
```

Весь контент лендинга — в `src/routes/index.tsx`, стили/токены — в `src/styles.css`.
